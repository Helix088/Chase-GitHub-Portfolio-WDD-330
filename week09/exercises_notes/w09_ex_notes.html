Notes

  1. The Browser Object Model is a collection of properties and methods that contain information about the browser and the computer screen.
  2. The properties can be accessed through the window object.
  3. You can use dialogs, get browser information, control the window, and get screen information.
  4. Each window object has a document object with properties and methods that deals with the loaded page.
  5. The user can user cookies, timing functions, intervals, and animation all within the window using these objects.
  6. HTML5 APIs can help a user gain access to cameras, batteries, geolocations, and graphics cards.
  7. HTML5 APIs are able to work in both old and new browsers with the help of shims and polyfills.
  8. AJAX does most of the work on the frontend of the webpage, whereas HTML APIs do momst of the work on the backend.
