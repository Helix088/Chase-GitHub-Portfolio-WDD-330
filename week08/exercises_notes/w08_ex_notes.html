Notes

  1. The CSS transform propetry lets change the appearence of any element on a page.
  2. This is accomplished by using transforming functions that allow the user to give elements some pizzazz.
  3. There are 4 different types of transforming functions which are: translate, scale, rotate, and skew.
  4. Transition the vvalues of the CSS properties to change over time producing simple animations.
  5. You must first declare the original state of the element in the default style declaration.
  6. Second you have to declare the final state of the transitioned element.
  7. Lastly you have to include the transition functions in your default style declaration using the transition properties.
  8. The properties are transition-property, transition-duration, transition-timing-function, and transition-delay.
  9. CSS animations allow you to control each step of an animation via keyframes, which gives them more flexibility over a transition.
  10. To animate an element in CSS, you first create a named animation, then attach it to an element in that element's property declaration block.
  11. Animations in themselves don't do anything; in order to animate an element, you'll need to associate the animation with that element.
  12. To create an animation yoiu will need to use the @keyframes rule and each keyframe should have its own nested declaration block.
  13. The animation properties are animation-name, animation-duration, animation-timing-function, animation-iteration-count, animation-direction, animation-delay, animation-fill-mode, and animation-play-state.
  14. Canvas allows the user to draw anything through JavaScript onto the HTML page.
  15. You create a canvas element: <canvas>Sorry the browser does not support Canvas!</canvas> it will only display the text if canvas is not supported in the browser.
  16. All drawing on the canvas happens through the Canvas JavaScript API.
  17. SVG stands for Scalable Vector Graphics and they are a specific file format that allows you to describe vector graphics using XML.
  18. You can draw a vector graphic by creating a svg drawing board like so: <svg></svg>
  19. The Drag and Drop API allows for certain elements to be draggable, and then specify what should happen when these elements are dragged over or dropped onto other elements on the page.
  20. First you must set the draggable attribute on any HTML elements you'd like to be draggable.
  21. Next add an event listener for the dragstart event on any draggable HTML element.
  22. Lastly, add an event listener for the dragover and drop events on any elements you want to have accept dropped items.
