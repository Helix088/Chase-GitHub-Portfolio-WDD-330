Notes

  • Forms help submitting information easier.

  • Object Oriented Programming helps bring real world objects into code.
  • This allows for programmers to make objects and use those objects to write programs that are focused on the object and it's children.
  • Modularizing JavaScript helps reduce the bloatedness of code by sectioning off smaller pieces of code into their own files.
